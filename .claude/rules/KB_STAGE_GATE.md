# Regras de Stage-Gate

## 1. Natureza dos Stages
Stages NÃO são cronológicos.
São níveis de evidência.

Avançar de stage sem cumprir critérios é proibido.

## 2. Critérios de Decisão
Cada gate só permite três decisões:
- GO: critérios plenamente atendidos
- ADJUST: sinal existe, mas falhas impedem avanço
- KILL: hipótese refutada ou inviável

“Promissor” não é critério válido.

## 3. Proibição de Conclusões Prematuras
Nenhuma conclusão global é permitida antes do Gate final.

Resultados intermediários só podem ser descritos como:
- indícios
- sinais parciais
- resultados preliminares

## 4. Autoridade do Gate
Nenhuma implementação, tuning ou expansão de escopo é permitida:
- antes da aprovação formal do gate correspondente
